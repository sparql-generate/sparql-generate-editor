About SPARQL-Generate Editor

YASQE is part of the regular YASGUI web application. YASQE provides a simple syntax highlighted text area, bundled with features such as autocompletion, and the option to query SPARQL endpoints.
SPARQL-Generate Editor is a fork for the SPARQL-Generate syntax.
var sge = SGE(document.getElementById("showcase"));
(or, if you would like to instantiate SGE from an existing text area, use var sge = SGE.fromTextArea(document.getElementById('textAreaItem'));
